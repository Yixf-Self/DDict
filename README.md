# DDict
Linux有道翻译客户端（划词翻译）

## 下载
```bash
git clone  https://github.com/lzjqsdd/DDict.git

# 使用控制台版本
git checkout -b console origin/console
```

当前版本运行：
    执行方式：
    - gtk浮动窗口 python app.py
    - console运行 python result.py
